# views

Folder ini dipakai untuk halaman utama aplikasi jika Vue Router sudah digunakan.

Contoh isi nanti:

- `DashboardView.vue`
- `LoginView.vue`
- `SensorHistoryView.vue`

Untuk onboarding awal, halaman masih berada di `App.vue`. Nanti ketika belajar routing, isi `App.vue` bisa dipindah ke `views/DashboardView.vue`.
