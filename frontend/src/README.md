# src

Folder ini adalah pusat kode frontend Vue.

Struktur ini dibuat mirip repo utama, tetapi tidak semua folder langsung dipakai. Tujuannya agar peserta mengenali peta project tanpa harus langsung mempelajari semua konsep sekaligus.

## Alur Baca Untuk Onboarding

1. `main.ts`: tempat Vue dipasang ke HTML.
2. `App.vue`: halaman demo yang sedang berjalan.
3. `services`: fungsi yang memanggil backend.
4. `api`: client hasil generate dari OpenAPI backend.
5. Folder lain seperti `components`, `router`, dan `views` dipelajari setelah alur dasar dipahami.

## Prinsip

Mulai dari yang berjalan dulu, lalu pecah ke folder yang lebih rapi ketika kode mulai membesar.
