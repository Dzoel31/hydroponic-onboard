# router

Folder ini dipakai untuk konfigurasi routing halaman.

Repo utama biasanya butuh router karena punya banyak halaman. Contohnya:

- dashboard
- halaman detail
- halaman login
- halaman settings

Untuk onboarding awal, aplikasi hanya punya satu halaman di `App.vue`, jadi Vue Router belum dipasang. Router berfungsi untuk mengatur navigasi antar halaman, sebagai contoh kita bisa cek pada repo `smart-hydroponic` yang sudah menggunakan Vue Router.
