# assets

Folder ini dipakai untuk aset yang di-import dari kode frontend.

Contoh isi:

- gambar ilustrasi
- icon lokal
- file CSS tambahan

Bedanya dengan `public`: file di `assets` biasanya dipakai lewat `import`, sedangkan file di `public` biasanya diakses langsung lewat URL.

Untuk onboarding awal, folder ini belum dipakai.
