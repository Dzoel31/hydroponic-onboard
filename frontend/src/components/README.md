# components

Folder ini dipakai untuk komponen Vue yang bisa dipakai ulang.

Contoh komponen yang nanti bisa dibuat:

- `MetricCard.vue`
- `ReadingTable.vue`
- `HydroponicForm.vue`

Untuk onboarding awal, UI masih ditulis di `App.vue` agar alurnya mudah dilihat dari satu file. Nanti setelah mulai bikin banyak halaman, kita bisa pisahkan UI ke komponen-komponen kecil di folder `components` agar lebih rapi dan mudah dipelihara.
